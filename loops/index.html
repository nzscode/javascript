Array Methods
Iterating through objects or arrays.
To iterate through the keys of an object using for ... in :
**For strings and arrays For...of is better.
Syntax
for (let key in object){
statement/return;
}
Example
let student1 = {
name: "Henry",
age: 5
};

for (let key in student1){
console.log(key);
};
Console
name
age

};

for (let key in student1){
console.log(student[key]);

To iterate through the values using for ... of :
Best for arrays, strings, generators, maps, sets
and user defined iterators.
NOT for objects
Syntax

Using for...in on arrays, objects or strings:
Syntax
let myArray = [1, 2, 3];

let myObj = {
key1: value1,
key2: value2
};

let myString = "dummy string"

for (let key in myArray){
console.log(`for...in array => ${myArray[key]}`);
};

for (let key in myObj){
console.log(`for...in Object=> ${myObj[key]}`);
};

for (let key in myString){
console.log(`for...in String=> ${myString[key]}`);
};
Example
let Apples = ["Red Delicious", "Bright Red", "Ontario"];

let Student = {
fName: "Jack",
lName: "Dawson",
friend: "Fabrizio"
}

let harryQuote = "Ron!"


for (let key in Apples){
console.log(`for...in array => ${Apples[key]}`);
};

for (let key in Student){
console.log(`for...in Object=> ${Student[key]}`);
};

for (let key in harryQuote){
console.log(`for...in String=> ${harryQuote[key]}`);
};
Console
for...in array => Red Delicious
for...in array => Bright Red
for...in array => Ontario

for...in Object=> Jack
for...in Object=> Dawson
for...in Object=> Fabrizio

for...in String=> R
for...in String=> o
for...in String=> n
for...in String=> !

Converting for to forEach:
on the forEach in the parameters, we can also include a second parameter which is for the index.
Syntax
let myArray = [1, 2, 3];

let anotherArray = []; for (let i = 0; i < myArray.length; i++){
anotherArray.push(myArray[i]);
};

OR


myArray.forEach ((item) => {
anotherArray.push(item);
};
Example
let Grades = [10, 8, 3];

let anotherArray = [];

for (let i = 0; i < Grades.length; i++){
anotherArray.push(Grades[i]);
};

OR


Grades.forEach ((item) => {
anotherArray.push(item);
};

console.log(anotherArray);
Console
(6) [10, 8, 3, 10, 8, 3]
