Template Strings

  • Template Strings use arguments to fill in parameters
    1. In a div, with the id: "ex1", and text "hiya"
    2. We will change the text from "hiya" to "hello" and a name 
let ex1 = document.getElementById("ex1");

let printName = function (name) {
    ex1.innerText = `Hello ${name}`;
};

printName("Alice");

    2. the return can be stored as a local variable within the scope of the function (i.e: resultAdd), or it can be returned, and then stored in an outer scope variable(i.e: rAdd)

    // let resultAdd = a + b;

    return a + b;

let rAdd = add(3, 2);

const multiply2 = function (resultAdd) {
    return 2 * resultAdd;

console.log(multiply2(rAdd));

Arrow Function

    1. Must have parentheses
    2. Function keyword is not used
    3. Inside the parentheses will be the parameters
    4. After the parentheses, comes an arrow
    5. Then comes a code block, followed by a semi-colon.
    6. for one parameter, the parentheses can be removed.
Code without Arrow Function 
const add = function(a, 3){
    return a + b;
};

console.log(add(2, 5));

//7
Code using Arrow Function 
const add = (a, 3) => {
    return a + b;
};

console.log(add(2, 5));

//7
Code with one parameter 
const add = a => {
    return a + 3;
};

console.log(add(2, 5));

//7
Code with one return statement 
const add = a => a + 3;


Immediately Invoked Function Expression

HTML
HTML
// Function - only invoked when called
function a() {
    console.log("hello");
}
a();
//Output:
//hello

// Function saved as variable - only invoked when called
var b = function () {
    console.log("hello coursera");
};

b();
//Output:
//hello coursera

// Immediately Invoked Function (IIFE)
(function () {
    console.log("hello udemy");
})();
//Output:
//hello udemy

// IIFE with variable
(function (fname) {
    console.log("good morning " + fname);
})("Claire");
//Output:
//good morning Claire

Print On Click Result as HTML

Example
HTML

<input type="text" name="fname" id="fname" >

Logging Mouse Position

  • Mouse Position will be logged using event.clientX & event.clientY
JS
document.querySelector("body").addEventListener("mousemove", function (event) {
    console.log("x: " + event.clientX);
    console.log("y: " + event.clientY);
});

Logging Mouse Position only when Shift Key is pressed

  • Will Only log the mouse position when shift key is also held down
Default CSS
document.querySelector("body").addEventListener("mousemove", function (event) {
    if (event.shiftKey === true) {
        console.log("x: " + event.clientX);
        console.log("y: " + event.clientY);
    }
});
