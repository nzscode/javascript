Labels with Break and Continue
Helps with loops when using break and continue.
Using continue with a label, is the same as using a break instead.
continue + label === break
Example 
                    let grades = [
                        [1,2,3],
                        [9,8,7],
                        [25,26,27]
                    ]

                    OuterLoop: for(let i = 0; < igrades.length; i++){
                        InnerLoop: for(let k =0; k < grades[i].length;k++){
                            console.log(grades[i][k]);
                            if (grades[i][k] === 8){
                                console.log("You found the 8");
                                continue OuterLoop;
                            }
                        }console.log("doing stuff");
                    }
                    console.log("~~~~~~")
                        //this code is ignored with continue OuterLoop, but not ignored with break

Functions call(), apply()
Accessing this inside functions using call() and apply()
Example 

                            function printThis(name){
                                console.log(name);
                                console.log(this);
                            }
                            
                            printThis.call("this", "Jenna");
                            //for apply you would put "Jenna" iside an array.
                                ex: printThis.apply("this", ["Jenna"]);
Console 
                     
                        Jenna
                        String {'this'}


Functions bind()
Must be assigned to a new variable since it creates a new function.
And unlike apply, there is no need to make the arguments into an array.
Example 

                        function printThis(name){
                            console.log(name);
                            console.log(this);
                        }
                        
                        let newFunction = printThis.bind("this", "Jenna");

                        newFunction();
Console 
                     
                        Jenna
                        String {'this'}
