Function Methods
bind()
Borrows funtion from another and uses it.
Using a Function from inside an object:
Syntax
let variableName = functionName.bind(obj);
invoke: console.log(functionName());
Example
let student1 = {
name: "Henry",
age: 5
};
let student2 = {
name: "Tessa",
age: 7
};

function greet(){
return `Hello ${this.name}, you are ${this.age} years old.`;
};

let greetStudent1 = greet.bind(student1);
let greetStudent2 = greet.bind(student2);

console.log(greetStudent1());
console.log(greetStudent2());
Console
Hello Henry, you are 5 years old.
Hello Tessa, you are 7 years old.


Using a global Function:
Syntax
let variableName = newObjectName.functionName.bind(objectWithFunctionName);
invoke: console.log(functionName());
Example
let student1 = {
name: "Henry",
age: 5,
greet: function(){
return `Hello ${this.name}, you are ${this.age} years old.`;
}
};

let student2 = {
name: "Tessa",
age: 7
};

let greetStudent1 = student1.greet.bind(student1);
let greetStudent2 = student1.greet.bind(student2);

console.log(greetStudent1());
console.log(greetStudent2());
Console
Hello Henry, you are five years old.
Hello Tessa, you are 7 years old.
