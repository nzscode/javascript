Objects

    Objects are non primitive data types that store key-value pairs. Non primitive data types are mutable data types that are not undefined, null, boolean, number, string, or symbol. Mutable means that the data can be changed after it is created. Here is the basic syntax for an object: 

    {
    key: value
}

    Objects are similar to arrays, except that instead of using indexes to access and modify their data, you access the data in objects through properties. Properties consist of a key and a value. The key is the name of the property, and the value is the data stored in the property. Here is an example of an object with a single property: 

    const obj = {
    name: "Quincy Larson"
};

    If the property name (key) of an object has a space in it, you will need to use single or double quotes around the name. Here is an example of an object with a property name that has a space: 

    const spaceObj = {
    "Space Name": "Kirk",
};

    If you tried to write a key without the quotes, it would throw an error

  • Accessing a property
    • Using Dot Notation

      • There are two ways to access the properties of an object: dot notation (.) and bracket notation ([]), similar to an array. Dot notation is what you use when you know the name of the property you're trying to access ahead of time. 

      object.property;
      Here is a sample of using dot notation ( . ) to read the name property of the developer object: 
      const developer = {
    name: "Jessica",
}
console.log(developer.name); //Jessica
    • Using Bracket Notation

      • The second way to access the properties of an object is bracket notation ([]). If the property of the object you are trying to access has a space in its name, you will need to use bracket notation. 

      objectName["property name"];

      Here is a sample of using bracket notation to read an object's property: 

      const spaceObj = {
    "Space Name": "Kirk",
};

spaceObj["Space Name"]; // "Kirk"

Selecting from an array of tags, then adding a class to each of them.

Situ :

I need to add a class to each H1 & H3 element of the html page, to turn them green. However, there are no classes attached to these tags.



Solu :
  1. I need to make an array of the tags I am looking for. 
    2. var Hs = ["h1", "h3"];
  2. I need to iterate through each of these tags, that are now part of an array(Node List) 
    3. for (let i = 0; i < Hs.length; i++) {
    var hs = Hs[i];
};
  3. I need to find all the tags that match the items in my array. 
    4. var h = document.querySelectorAll(hs);
  4. I need to make a function that will toggle on the classes I want to show up on the tags. 
    5. function colorRed(element) {
    element.classList.toggle("oswald-font");
    element.classList.toggle("green");
}
  5. I need to toggle a class on to each of the tags.
    6. h.forEach(colorRed);
    var Hs = ["h1", "h3"];
for (let i = 0; i < Hs.length; i++) {
    var hs = Hs[i];
    var h = document.querySelectorAll(hs);

    function colorRed(element) {
element.classList.toggle("oswald-font");
element.classList.toggle("green");
    }
    h.forEach(colorRed);
}
